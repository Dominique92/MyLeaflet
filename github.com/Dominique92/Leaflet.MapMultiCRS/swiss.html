Using GeoAdmin WMTS tiles with the popular Leaflet library (II)

Now using the Proj4Leaflet library it is even easier to display geo.admin.ch WMTS tiles into the Leaflet framework, which is normally sperical mercator.

Using Leaflet v0.5, we can now display GeoJSON data, in this case, national parks, with nice popup if you click on theme features


View the leaflet-projection-v2.js source to see how this is done.
