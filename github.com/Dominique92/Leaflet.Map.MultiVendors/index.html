Leaflet extension for multi vendors maps & projections ( OSM, Maps.Refuges.Info, Thunder Forest, Refuges.info, Hike & Bike, ClycleMap, Outdoors & Bike, IGN France, SwissTopo, IDEE España, IGM Italie, OS Great Britain, Kompass Austria, Bing Microsoft, Google maps, ...)

Hover the switcher icon on top right of the map & click on the required layer.

Sources & docs on Github Dominique92/Leaflet.Map.MultiVendors

This demo runs on Leaflet V ➠ See demo on Leaflet V1.0
