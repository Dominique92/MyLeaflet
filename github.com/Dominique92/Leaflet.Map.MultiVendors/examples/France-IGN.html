Leaflet extension for France IGN maps

* Note: To run on localhost you need to ask form a development key to: http://api.ign.fr

* Note: IGN Topo & Classic doesn't work on localhost

Sources & docs on Github Dominique92/Leaflet.Map.MultiVendors/blob/master/layers/IGN.js

This demo runs on Leaflet and is also supported on Leaflet V1.0
