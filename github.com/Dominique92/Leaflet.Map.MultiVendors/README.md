# Leaflet.Map.MultiVendors
Leaflet extension for multi vendors maps & projections
```
- OSM, Maps.Refuges.Info, Thunder Forest, Refuges.info, Hike & Bike, ClycleMap, Outdoors & Bike
- IGN France
- SwissTopo
- IDEE Espa√±a
- IGM Italie
- OS Great Britain
```
