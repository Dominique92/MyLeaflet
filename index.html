Leaflet extension for POI & tracks management on multi vendors maps.

This demo runs on Leaflet V

See the javascript code of this demo here and here.

To get a local demo, copy the html source of this page on a local .html file & call it from your brother (no PHP required).

If some maps (Swiss) doesn't display on this server because of a map licence problems, copy this file on your localhost & run it on the dev map license (see providers docs).

Sources & docs on Github: Dominique92/MyLeaflet

Other plugins demos: GeoJSON.Ajax   Map.MultiVendors   Permalink.Cookies   Marker.coordinates   Draw.plus  

Specific layers: France IGN,   SwissTopo,   Spain IDEE,   Italy IGM,   OSM overpass

For Leaflet V0.7 you can have a look on the unsupported old package.
